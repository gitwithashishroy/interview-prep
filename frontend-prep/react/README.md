# React Interview Questions

## Topics Covered
- Component Lifecycle
- Hooks (useState, useEffect, useCallback, useMemo, useRef)
- Custom Hooks
- Context API
- Performance Optimization
- Error Boundaries
- Reconciliation & Virtual DOM

## Naming Convention
`ComponentName.tsx` or `use-hook-name.ts`

**Examples:**
- `Counter.tsx`
- `use-fetch.ts`
- `TodoList.tsx`
