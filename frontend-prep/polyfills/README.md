# JavaScript Polyfills

## Common Polyfills
- Array methods (map, filter, reduce, forEach, find, findIndex)
- Promise.all, Promise.race, Promise.allSettled
- Function methods (call, apply, bind)
- Object methods (assign, create, keys, values)
- Array.flat, Array.flatMap
- debounce, throttle

## Naming Convention
`methodName.polyfill.js`

**Examples:**
- `map.polyfill.js`
- `bind.polyfill.js`
- `promise-all.polyfill.js`
